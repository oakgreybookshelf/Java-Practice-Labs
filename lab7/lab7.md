This lab uses interfaces, abstract classes, the Random class, and generic types to create a gambling game. Different game types (lotto, slot machine, and dice) can be instantiated into the GameOfChance class to allow varied gameplay.  The GameOfChance class has built-in functions to check for a winner using different logic depending on what game is currently being played. The lab assignment uses the strategy design pattern to implement different game logic depending on which type of game the user is playing.
